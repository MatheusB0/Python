Listas

Listas são usadas para armazenar varíos itens em uma variável.

Listas são um dos quatros tipos de dados internos do Python usados para armazenar coleções de dados; os outros três são Tuple, Set e Dictionary, todos com qualidades e usos diferentes.

As listas são criadas usando colchetes:

Exemplo:
    lista =["maça", "banana", "cereja"]
    print(lista)

Saida:
    ['maça', 'banana', 'cereja']

Itens da lista

Os itens de uma lista são ordenados, alteráveis e permitem valores duplicados.

os itens da lista são indexados, o primeiro tem indice [0], o segundo item tem indice [1] e assim por diante.

Ordenado

Quando dizemos que as listas são ordenadas, isso significa que os itens têm uma ordem definida e essa ordem não mudará

Observação: existem alguns métodos de lista que alteram a ordem, mas, em geral, a ordem dos itens não muda.

Mutável

A lista é mutável, o que significa que podemos alterar, adicionar e remover itens de uma lista depois que ela foi criada.

Permitir Duplicatas

Como as listas são indexadas, elas podem ter itens com o mesmo valor

Exemplo:
    lista = ["maça","banana","cereja","maça","cereja"]
    print(lista)

Saida:
    ['maça', 'banana', 'cereja', 'maça', 'cereja']

Comprimento da lista

Para determinar quantos itens uma lista possui, use a len()função.

Exemplo:
    lista = ["maça","banana","cereja"]
    print(len(lista))

Saida:
    3

Itens da lista - Tipos de dados

Os itens de uma lista podem ser de qualquer tipo de dados.

Exemplo:
    lista1 = ["maça", "banana", "cereja"]
    lista2 = [1, 5, 7, 12, 3]
    lista3 = [True, False, False, True]

Saida:
    ['maça', 'banana', 'cereja']
    [1, 5, 7, 12, 3]
    [True, False, False, True]

Uma lista pode contem diferentes tipos de dados.

Exemplo:
    lista1 = ["abc", 34, True, 40]

Saida:
    ['abc', 34, True, 40]

type()

Da perspectiva do Python, listas são definidas como objetos com o tipo de dados 'list':

<class 'list'>
Exemplo:
    lista = ["maça", "banana", "cereja"]
    print(type(lista))

Saida:
    <class 'list'>

O construtor list()

Também é possível usar o construtor list() ao criar uma nova lista.

Exemplo:
    #Observe que ultlizamos colchetes duplos
    lista = list(("maça", "banana", "cereja")) 
    print(lista)

Saida:
    ['maça', 'banana', 'cereja']
