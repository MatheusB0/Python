Os dicionários são uma estrutura de dados fundamental em Python, permitindo armazenar pares chave-valor. Eles são muito úteis para organizar e acessar dados de forma eficiente.

Criando um Dicionário

Um dicionário é criado usando chaves {} e os elementos são definidos como chave: valor:

# Exemplo de dicionário
            meu_dicionario = {
                "nome": "Carlos",
                "idade": 25,
                "cidade": "São Paulo"
            }

Acessando Valores

Podemos acessar valores no dicionário usando a chave correspondente:

print(meu_dicionario["nome"])  # Saída: Carlos

Para evitar erros caso a chave não exista, usamos o método get():

print(meu_dicionario.get("profissao", "Chave não encontrada"))

Modificando um Dicionário

Podemos adicionar ou modificar elementos atribuindo um valor a uma chave:


            meu_dicionario["idade"] = 26  # Modifica a idade
            meu_dicionario["profissao"] = "Engenheiro"  # Adiciona uma nova chave

Removendo Elementos

Para remover um item do dicionário:

del meu_dicionario["cidade"]  # Remove a chave 'cidade'

O método pop() também pode ser usado:

idade = meu_dicionario.pop("idade")  # Remove e retorna o valor associado

Iterando sobre um Dicionário

Podemos percorrer um dicionário com um loop for:


            for chave, valor in meu_dicionario.items():
                print(f"{chave}: {valor}")

Métodos úteis de Dicionários

Python fornece vários métodos para manipular dicionários:


            meu_dicionario.keys()    # Retorna todas as chaves
            meu_dicionario.values()  # Retorna todos os valores
            meu_dicionario.items()   # Retorna pares chave-valor
            meu_dicionario.clear()   # Remove todos os elementos

Quando Usar um Dicionário?

  • Quando precisamos mapear relações entre chaves e valores.
  • Para armazenar configurações ou dados estruturados.
  • Quando precisamos acessar elementos rapidamente através de uma chave.
