Introdução ao Python

O Python é uma linguagem de programação de alto nível, interpretada e de fácil aprendizado, ideal para quem está começando a programar. Além disso, o Python é uma linguagem muito versátil, podendo ser usada para desenvolvimento web, automação, análise de dados, inteligência artificial, e muito mais.

Como instalar o Python

Para começar a programar em Python, você precisa instalá-lo em seu computador. Acesse o site oficial do Python: python.org e baixe a versão mais recente do Python para o seu sistema operacional.

Primeiro Programa

Vamos começar com um exemplo clássico: o famoso "Olá, Mundo!". No Python, isso pode ser feito com o comando print().


print("Olá, Mundo!")

Quando você rodar esse código, o Python exibirá a mensagem "Olá, Mundo!" na tela. Simples assim!

Comentários

Comentários são úteis para documentar seu código e explicar o que ele faz. Em Python, os comentários começam com o símbolo #.


# Este é um comentário
print("Este código vai exibir esta mensagem")

Indentação

Python usa a indentação (espaços ou tabulação) para organizar o código. Ao escrever funções, loops ou condicionais, é importante manter a indentação correta para garantir que o código funcione.


if True:
    print("Esta linha de código está dentro do bloco if")
