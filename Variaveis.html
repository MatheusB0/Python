O que são Variáveis?

Variáveis são espaços na memória onde podemos armazenar valores temporários. Essas variáveis podem ser usadas para manipular, calcular e passar informações ao longo do seu código.

Por exemplo, quando você cria uma variável para armazenar a idade de uma pessoa, você está guardando um valor numérico (um inteiro) nessa variável, que pode ser usado para realizar outras operações ou exibir para o usuário.

Como Criar e Usar Variáveis

Em Python, não é necessário declarar o tipo da variável antes de usá-la, pois o tipo é inferido automaticamente com base no valor atribuído.


            idade = 25  # A variável 'idade' recebe um valor inteiro
            nome = "João"  # A variável 'nome' recebe uma string

No exemplo acima, a variável idade está armazenando um número inteiro e a variável nome está armazenando uma cadeia de caracteres (string).

Tipos de Dados em Python

Em Python, existem vários tipos de dados que você pode usar. Os tipos de dados mais comuns são:

  1. Inteiro (int): Números inteiros, sem ponto decimal.
  2. Ponto flutuante (float): Números com ponto decimal.
  3. String (str): Texto ou sequências de caracteres.
  4. Booleano (bool): Valores lógicos, que podem ser True ou False.

Explicação dos Tipos de Dados

int: O tipo int (inteiro) é usado para representar números inteiros, como 1, 100, -50, etc. Não é possível ter valores decimais com esse tipo de dado.

Exemplo: numero = 10

float: O tipo float é usado para representar números com ponto flutuante, ou seja, números que possuem casas decimais.

Exemplo: pi = 3.14159

str: O tipo str (string) é utilizado para armazenar sequências de caracteres, ou seja, texto. As strings podem ser envolvidas por aspas simples ou duplas.

Exemplo: nome = "Carlos"

bool: O tipo bool (booleano) é usado para armazenar valores lógicos. Ele pode ter apenas dois valores possíveis: True (verdadeiro) ou False (falso).

Exemplo: flag = True

Conversão de Tipos

  1. int(): Converte um valor para inteiro.
  2. float(): Converte um valor para ponto flutuante.
  3. str(): Converte um valor para string.
  4. bool(): Converte um valor para booleano.

Exemplo de conversão de tipos:

# Convertendo string para inteiro
numero = int("10")

# Convertendo inteiro para string
texto = str(100)

# Convertendo valor inteiro para booleano
booleano = bool(1)  # 1 é considerado como True

Usando Tipos em Operações

Operações com int e float: Você pode realizar operações matemáticas com inteiros e flutuantes, como soma, subtração, multiplicação e divisão.

Exemplo:
a = 5
b = 2.5
soma = a + b  # 5 + 2.5 = 7.5

Operações com str: Strings podem ser concatenadas com o operador + e repetidas com o operador *

Exemplo:
saudacao = "Olá, " + "Maria!"  # Resultado: "Olá, Maria!"
repeticao = "Olá! " * 3  # Resultado: "Olá! Olá! Olá! "

Operações com bool: Os valores booleanos podem ser usados com operadores lógicos, como and, or, e not.

Exemplo:
a = True
b = False
resultado = a and b  # Resultado: False
