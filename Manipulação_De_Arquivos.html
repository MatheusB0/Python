1. Manipulação de Strings

Strings são sequências de caracteres usadas para armazenar texto. Em Python, são representadas por aspas simples ('texto') ou duplas ("texto").

1.1 Operações Básicas


            s1 = "Hello"
            s2 = "World"
            
            # Concatenação (Juntar Strings)
            resultado = s1 + " " + s2  # "Hello World"
            
            # Repetição
            repetida = s1 * 3  # "HelloHelloHello"
            
            # Indexação (Acessar um caractere)
            primeira_letra = s1[0]  # "H"
            
            # Fatiamento (Pegar parte da string)
            pedaco = s1[1:4]  # "ell"

1.2 Métodos Mais Usados


            texto = "  Python é incrível!  "

            # Converter para maiúsculas e minúsculas
            print(texto.upper())   # "  PYTHON É INCRÍVEL!  "
            print(texto.lower())   # "  python é incrível!  "
            
            # Remover espaços extras
            print(texto.strip())  # "Python é incrível!"
            
            # Substituir palavras
            print(texto.replace("incrível", "fantástico"))  
            # "  Python é fantástico!  "
            
            # Dividir uma string em uma lista
            palavras = texto.split()  # ['Python', 'é', 'incrível!']

2. Manipulação de Arquivos

Python permite abrir, ler e escrever arquivos de maneira simples.

2.1 Abrindo um Arquivo


        with open("arquivo.txt", "r") as f:
            conteudo = f.read()
        print(conteudo)  # Exibe o conteúdo do arquivo

"r": leitura

"w": escrita (apaga o conteúdo anterior)

"a": acrescenta ao final

"b": modo binário

2.2 Lendo Arquivos Linha por Linha


            with open("arquivo.txt", "r") as f:
                for linha in f:
                    print(linha.strip())

2.3 Escrevendo em um Arquivo


            with open("arquivo.txt", "w") as f:
            f.write("Aprendendo Python!\n")

Se quiser adicionar sem apagar o conteúdo:


            with open("arquivo.txt", "a") as f:
            f.write("Adicionando mais texto.\n")

2.4 Trabalhando com Arquivos CSV


            import csv
            with open("dados.csv", "r") as f:
                leitor = csv.reader(f)
                    for linha in leitor:
                        print(linha)

Escrever em CSV:


            with open("dados.csv", "w", newline='') as f:
            escritor = csv.writer(f)
            escritor.writerow(["Nome", "Idade"])
            escritor.writerow(["Alice", 30])

2.5 Trabalhando com JSON


            import json
            with open("dados.json", "r") as f:
                dados = json.load(f)
            print(dados)

Salvar dados em JSON:


            dados = {"nome": "Alice", "idade": 30}
            with open("dados.json", "w") as f:
                json.dump(dados, f, indent=4)

2.6 Tratamento de Erros ao Trabalhar com Arquivos


            try:
                with open("arquivo_inexistente.txt", "r") as f:
                    conteudo = f.read()
            except FileNotFoundError:
                print("Arquivo não encontrado!")
