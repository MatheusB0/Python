Listas Aninhadas

Listas aninhadas são listas dentro de outras listas, ou seja, uma lista pode armazenar outras listas como seus elementos. Isso é útil para representar dados mais complexos, como matrizes ou tabelas.

Como Criar uma Lista Aninhada?

Existem duas formas principais de criar listas aninhadas:

1. Criando Diretamente com Colchetes

Exemplo:
lista_aninhada = [[1, 2, 3], ["a", "b", "c"], [True, False]]
print(lista_aninhada)

Saída:
[[1, 2, 3], ['a', 'b', 'c'], [True, False]]

Representação Visual:

lista_aninhada
├── [1, 2, 3]   <- Sublista 1
├── ['a', 'b', 'c'] <- Sublista 2
└── [True, False] <- Sublista 3

2. Usando o Construtor list()

Exemplo:
lista_aninhada = list([[1, 2, 3], ["a", "b", "c"], [True, False]])
print(lista_aninhada)

Saída:
[[1, 2, 3], ['a', 'b', 'c'], [True, False]]

Representação Visual:

lista_aninhada
├── [1, 2, 3]   <- Sublista 1
├── ['a', 'b', 'c'] <- Sublista 2
└── [True, False] <- Sublista 3

Como Acessar Itens em Listas Aninhadas?

Você pode acessar itens em listas aninhadas usando múltiplos índices. O primeiro índice acessa a lista externa e o segundo índice acessa os elementos dentro da sublista.

Exemplo:
lista_aninhada = [[1, 2, 3], ["a", "b", "c"], [True, False]]
print(lista_aninhada[0][0])  # Acessa o primeiro item da primeira sublista (1)

Saída:
1

Resumo

Listas aninhadas permitem que você armazene coleções de listas, facilitando o gerenciamento de dados em múltiplas dimensões. Você pode criar listas aninhadas diretamente com colchetes ou utilizando o construtor list(). Acesse os itens internos usando múltiplos índices.
