Dicionários aninhados são dicionários dentro de dicionários. Eles são úteis para organizar dados hierárquicos ou estruturados de forma complexa.

Criando um Dicionário Aninhado

Podemos criar um dicionário onde os valores também são dicionários:


            empresa = {
                "nome": "Tech Corp",
                "departamentos": {
                    "TI": {"funcionarios": 10, "orçamento": 50000},
                    "RH": {"funcionarios": 5, "orçamento": 20000}
                }
            }

Acessando Elementos

Para acessar um valor dentro de um dicionário aninhado, basta encadear as chaves:

print(empresa["departamentos"]["TI"]["funcionarios"])  # Saída: 10

Modificando um Dicionário Aninhado

Podemos modificar os valores normalmente:

empresa["departamentos"]["RH"]["funcionarios"] = 7

Também podemos adicionar novos subdicionários:

empresa["departamentos"]["Financeiro"] = {"funcionarios": 8, "orçamento": 30000}

Removendo Elementos

Podemos remover um subdicionário usando del ou pop():

del empresa["departamentos"]["RH"]
financeiro = empresa["departamentos"].pop("Financeiro")

Iterando sobre Dicionários Aninhados

Para percorrer um dicionário aninhado, utilizamos loops aninhados:

             for depto, dados in empresa["departamentos"].items():
                    print(f"Departamento: {depto}")
                        for chave, valor in dados.items():
                            print(f"  {chave}: {valor}")

Quando Usar um Dicionário Aninhado?

  • Quando os dados possuem uma estrutura hierárquica.
  • Para representar relações complexas entre elementos.
  • Quando precisamos armazenar múltiplos conjuntos de dados relacionados.

Dicionários aninhados tornam o armazenamento e a manipulação de dados mais organizados e eficientes.
