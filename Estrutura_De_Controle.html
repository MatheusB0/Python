1. Condicional: `if`, `elif` e `else`

As estruturas condicionais permitem que o programa tome decisões com base em condições lógicas.

Estrutura Básica:

 
if condição:
    # código a ser executado se a condição for verdadeira
elif outra_condição:
    # código a ser executado se a outra_condição for verdadeira
else:
    # código a ser executado se nenhuma das condições for verdadeira

Exemplo:


idade = 20

if idade >= 18:
    print("Você é maior de idade.")
elif idade >= 12:
    print("Você é um adolescente.")
else:
    print("Você é uma criança.")

Este código verifica a idade de uma pessoa e imprime a classificação correspondente.

2. Laços de Repetição

Laços de repetição são usados para executar um bloco de código várias vezes.

2.1 Laço `for`

O laço `for` é usado quando sabemos de antemão quantas vezes o código precisa ser repetido. Ele é geralmente usado para iterar sobre sequências (como listas ou strings), mas pode ser usado de maneira mais simples.

Exemplo:


for i in range(5):
    print(i)

Este código imprime os números de 0 a 4, utilizando o `range(5)` para gerar os números de 0 a 4.

2.2 Laço `while`

O laço `while` é usado quando não sabemos quantas vezes o código será repetido, mas a repetição continua enquanto uma condição for verdadeira.

Exemplo:


contador = 0

while contador < 5:
    print(contador)
    contador += 1

Este código imprime os números de 0 a 4. O laço continua até a condição `contador < 5` ser falsa.

3. Comandos de Controle

O Python oferece comandos que ajudam a controlar a execução dos laços e das estruturas condicionais:

3.1 Comando `break`

O comando `break` interrompe um laço antes que ele termine normalmente.

Exemplo:


for i in range(10):
    if i == 5:
        break
    print(i)

O laço imprimirá os números de 0 a 4 e interromperá quando `i` for igual a 5.

3.2 Comando `continue`

O comando `continue` pula a iteração atual e continua com a próxima iteração do laço.

Exemplo:


for i in range(5):
    if i == 3:
        continue
    print(i)

Este código ignora o número 3 e imprime os números 0, 1, 2, 4.

3.3 Comando `pass`

O comando `pass` não faz nada. Ele é usado como um placeholder quando a sintaxe exige um bloco de código, mas você não quer fazer nada.

Exemplo:


for i in range(5):
    if i == 3:
        pass  # Não faz nada quando i for igual a 3
    print(i)

A saída desse código será 0, 1, 2, 3, 4, mas com o `pass`, o número 3 não será tratado de maneira especial.
